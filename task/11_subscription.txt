SUBSCRIPTION PAGE (/subscription)
================================================================

File: app/subscription/page.tsx
Route: /subscription
Auth Required: No (but shows authenticated-style layout)
Type: Client component ("use client")

================================================================
DESCRIPTION
================================================================

Subscription management page showing current plan details, usage analytics,
billing history, and quick action links. Uses the SubscriptionStatus widget
in the sidebar.

NOTE: Currently uses HARDCODED MOCK DATA — not connected to payment or
subscription API endpoints.
