================================================================
DIRECTORY PURPOSE SUMMARY
================================================================

app/            Next.js App Router pages. Each folder = a route. Uses "use client" directive on all pages (client-side rendering).
components/     Reusable React components shared across pages. Split into custom app components and ui/ (shadcn primitives).
components/ui/  shadcn/ui component library — 40+ headless UI primitives built on Radix UI.
hooks/          Custom React hooks for auth state, user profile, mobile detection, and toasts.
lib/            Core utilities — API client with endpoint definitions, user data normalization, and Tailwind merge helpers.
public/         Static files served at root URL.
styles/         Additional CSS files.

================================================================
CONFIGURATION
================================================================

Path Alias: @/* maps to ./* in tsconfig.json
Dark Mode: html className="dark" (class-based, not media query)
Domain: saksin.online (redirect configured in next.config.mjs)
Font: Inter (Google Fonts via next/font)

================================================================
ENVIRONMENT VARIABLES
================================================================

NEXT_PUBLIC_API_URL              Main backend API base URL (default: http://localhost:8000)
NEXT_PUBLIC_API_URL1             Chatbot "Get" service base URL (default: http://localhost:8005)
NEXT_PUBLIC_GOOGLE_CLIENT_ID     Google OAuth 2.0 Client ID

================================================================
GLOBAL CSS (globals.css)
================================================================

- Custom scrollbar styling
- .glass class (glassmorphism effect)
- .glow-emerald / .glow-blue (glow effects)

- animate-pulse-glow (pulsing glow animation)
- Tailwind directives (@tailwind base, components, utilities)
