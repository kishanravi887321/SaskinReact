================================================================
ROLE SELECTION PAGE (/roles)
================================================================

File: app/roles/page.tsx
Route: /roles
Auth Required: No
Type: Client component ("use client")

================================================================
DESCRIPTION
================================================================

A grid page where users select their interview role before starting a practice
session. Each role card links to /interview/[role.id] with the dynamic role
parameter.

================================================================
AVAILABLE ROLES
================================================================

ID            Label              Icon
sde           Software Engineer  💻
pm            Product Manager    📱
analyst       Data Analyst       📊
consultant    Business Consultant 💼
designer      UX/UI Designer     🎨
devops        DevOps Engineer    ⚙️

================================================================
COMPONENTS USED
================================================================

UI Components:
- Card, CardContent
- Button

Libraries:
- framer-motion (card hover/entrance animations)
- lucide-react (icons)
- next/link (navigation to /interview/[role])

================================================================
API CALLS
================================================================

None — roles are hardcoded in the component.
